# JavaScript Project with Vitest

Простой проект на JavaScript с использованием Vitest для тестирования.

## Установка

```bash
npm install
```

## Запуск тестов

Для запуска тестов в режиме наблюдения:
```bash
npm test
```

Для запуска тестов с отчетом о покрытии:
```bash
npm run test:coverage
```

## Структура проекта

- `src/math.js` - основные математические функции
- `src/math.test.js` - тесты для математических функций
